# docker-for-rails

Rails動かす用のDocker環境です

## How to use

1. `mv docker-for-rails YOUR_PROJECT_NAME`
2. `cd YOUR_PROJECT_NAME`
3. `cp env-example .env`
4. .envに値を書き込む
5. `mkdir YOUR_PROJECT_NAME
6. `docker-compose up -d`
7. `make build`
